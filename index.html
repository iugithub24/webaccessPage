Web Accessibility

The WAI Initiative

The Web Accessibility Initiative (WAI) is an effort by the World Wide Web Consortium (W3C) to improve the accessibility of the web for people with disabilities. It develops guidelines and resources to promote accessible web design and content.

The 4 Tenets of Web Accessibility

Web accessibility is based on four core principles known as the "POUR" principles: Perceivable, Operable, Understandable, and Robust.

Accessibility Guidelines

WCAG-EM (Web Content Accessibility Guidelines Evaluation Methodology) is a framework developed by WAI for evaluating web accessibility. It provides a structured approach to assessing how well websites conform to the Web Content Accessibility Guidelines (WCAG).

Semantic HTML Usage

Semantic HTML tags (e.g. <header>, <nav>, <article>, <footer>) provide meaning and structure to web content. Using semantic tags enhances accessibility by providing clearer cues to assistive technologies about the purpose and structure of the content.

ARIA Landmarks

ARIA (Accessible Rich Internet Applications) attributes enhance accessibility for all users.
